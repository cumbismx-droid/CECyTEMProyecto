Información de la Materia

La materia de Cultura Digital tiene como objetivo que los estudiantes comprendan el uso responsable de la tecnología, el internet y las herramientas digitales en la vida cotidiana y escolar.

Se analizan temas como la identidad digital, la seguridad en línea, la comunicación digital y el impacto de la tecnología en la sociedad.

Primer Parcial

¿Qué es la Cultura Digital?

La cultura digital se refiere a la forma en que las personas utilizan la tecnología digital para comunicarse, aprender, trabajar y convivir dentro de la sociedad.

Elementos de la cultura digital

  • Uso de internet y redes sociales.
  • Comunicación digital.
  • Acceso a la información.
  • Identidad digital.
  • Uso responsable de la tecnología.

¿Qué es la Cultura Digital?

La cultura digital es el conjunto de conocimientos, habilidades, valores y prácticas que permiten a las personas interactuar de manera responsable y eficiente con la tecnología digital.

Incluye el uso de dispositivos electrónicos, internet, redes sociales y herramientas digitales para comunicarse, aprender y resolver problemas en la vida diaria.

Importancia de la Cultura Digital

  • Facilita la comunicación entre personas.
  • Permite acceder a información rápidamente.
  • Mejora el aprendizaje y la educación.
  • Ayuda a usar la tecnología de forma responsable.
  • Fomenta el pensamiento crítico.

Uso responsable de la tecnología

Usar la tecnología de manera responsable implica respetar a los demás, cuidar la información personal y evitar el mal uso de las redes sociales y dispositivos digitales.

  • No compartir información personal.
  • Respetar opiniones de otras personas.
  • No realizar ciberacoso.
  • Usar contraseñas seguras.
  • Verificar la información antes de compartirla.

¿Qué es la investigación?

La investigación es un proceso mediante el cual se busca obtener información confiable sobre un tema, con el objetivo de aprender, resolver problemas o ampliar conocimientos.

Tipos de investigación

Investigación documental

Se basa en la búsqueda y análisis de información obtenida de libros, revistas, páginas web, artículos y documentos digitales.

Investigación de campo

Consiste en recopilar información directamente del lugar donde ocurren los hechos, mediante encuestas, entrevistas u observación.

Investigación experimental

Se realiza cuando el investigador experimenta y modifica variables para observar resultados, generalmente en laboratorios o prácticas controladas.

Investigación descriptiva

Su objetivo es describir características de una situación, persona o fenómeno sin modificarlo.

Investigación exploratoria

Se utiliza cuando el tema es poco conocido y se busca obtener información general para comprenderlo mejor.

Importancia de la investigación

  • Permite adquirir nuevos conocimientos.
  • Ayuda a resolver problemas.
  • Fomenta el pensamiento crítico.
  • Mejora el aprendizaje escolar.
  • Permite tomar decisiones informadas.

Segundo Parcial

Lenguaje HTML y CSS

¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado que se utiliza para crear la estructura de las páginas web. Permite organizar el contenido mediante etiquetas como títulos, párrafos, imágenes y enlaces.

Etiquetas básicas de HTML

  • <html>: Define el inicio del documento.
  • <head>: Contiene información del documento.
  • <title>: Define el título de la página.
  • <body>: Contiene el contenido visible.
  • <h1> a <h6>: Títulos.
  • <p>: Párrafos.
  • <img>: Imágenes.
  • <a>: Enlaces.
  • <table>: Tablas.
  • <tr>: Filas de una tabla.
  • <td>: Celdas de una tabla.

¿Qué es CSS y para qué se utiliza?

CSS (Cascading Style Sheets) es un lenguaje de estilos que se utiliza para dar presentación a los documentos escritos en HTML o XML. Permite cambiar colores, tamaños, fuentes, márgenes y posiciones de los elementos.

Estructura básica de CSS

Una regla CSS se compone de un selector y un bloque de declaraciones, donde se definen las propiedades y valores que se aplican a un elemento.

¿Qué es un selector en CSS?

Un selector en CSS es la forma de indicar a qué elemento HTML se le aplicarán los estilos. Existen diferentes tipos de selectores.

Selector de etiqueta

Se utiliza para aplicar estilos a todas las etiquetas del mismo tipo. Ejemplo: p, h1, div.

Selector de clase

Se utiliza para aplicar estilos a uno o más elementos HTML que tienen el mismo atributo class. Se representa con un punto (.).

Ejemplo: .titulo

Selector de ID

Se utiliza para aplicar estilos a un solo elemento HTML que tiene un identificador único. Se representa con el símbolo #.

Ejemplo: #contenedor

¿Qué es una hoja de estilo?

Una hoja de estilo es un conjunto de reglas CSS que se utilizan para definir la apariencia visual de una página web. Puede estar dentro del archivo HTML o en un archivo externo.

Ventajas de usar CSS

  • Mejora la apariencia de la página.
  • Permite separar contenido y diseño.
  • Facilita el mantenimiento del sitio web.
  • Hace que las páginas sean más atractivas.
  • Permite reutilizar estilos.

Componentes principales de una computadora

Una computadora está formada por dos partes fundamentales: el hardware y el software, que trabajan juntos para permitir el funcionamiento del equipo.

Hardware

El hardware se refiere a todos los componentes físicos y tangibles de una computadora, es decir, las partes que se pueden tocar y ver.

  • CPU: Es el cerebro de la computadora, procesa la información.
  • Placa base: Conecta todos los componentes del sistema.
  • Memoria RAM: Almacena datos temporales mientras se ejecutan programas.
  • Disco duro o SSD: Guarda información de forma permanente.
  • Fuente de poder: Proporciona energía a todos los componentes.
  • Gabinete: Protege y contiene los componentes internos.
  • Monitor: Muestra la información en pantalla.
  • Teclado y mouse: Permiten la interacción del usuario.

CPU (Unidad Central de Procesamiento)

La CPU es el componente encargado de ejecutar instrucciones y realizar cálculos. Existen diferentes marcas y modelos.

  • Intel: Procesadores como Core i3, i5, i7 y i9.
  • AMD: Procesadores como Ryzen 3, Ryzen 5 y Ryzen 7.

Software

El software es el conjunto de programas y aplicaciones que permiten que la computadora funcione y realice tareas específicas.

  • Sistema operativo: Windows, Linux, macOS.
  • Programas: Word, Excel, navegadores web.
  • Aplicaciones: Apps de comunicación, diseño o estudio.

Diferencia entre programa y software

Un programa es una aplicación específica que realiza una tarea, mientras que el software es el conjunto de programas que permiten el funcionamiento general del sistema.

Conceptos de seguridad digital

  • Phishing: Técnica para engañar a las personas y robar información personal.
  • Spam: Mensajes no deseados enviados de forma masiva.
  • Scam: Fraude que busca engañar para obtener dinero o datos.
  • Malware: Software malicioso diseñado para dañar dispositivos.
  • Ransomware: Bloquea el sistema y pide rescate.
  • Catfishing: Crear identidades falsas en internet.
  • Grooming: Acoso de adultos hacia menores en línea.

Importancia de la cultura digital

Conocer estos conceptos ayuda a usar la tecnología de forma segura, responsable y consciente, evitando riesgos y protegiendo la información personal.

Tercer Parcial

Medios de información

Los medios de información son herramientas que se utilizan para comunicar, informar y transmitir conocimientos a las personas. Su función principal es mantener informada a la sociedad sobre hechos relevantes a nivel local, nacional e internacional.

Los medios de información deben ser claros, veraces y accesibles, respetando el derecho de todas las personas a estar informadas.

Tipos de medios de información

  • Televisión
  • Radio
  • Periódicos
  • Revistas
  • Internet
  • Redes sociales

Elementos de los medios informativos

  • Ciudad o país
  • Autor
  • Fuente de información
  • Fecha de publicación
  • Dirección electrónica
  • Nombres de lectores
  • Correo electrónico
  • Especialización del medio
  • Otros datos relevantes

Importancia de los medios de comunicación

Los medios de comunicación influyen en la sociedad y en la forma de pensar de las personas, ya que permiten conocer sucesos actuales y facilitan el acceso a la información.

Internet

Internet es una red mundial que conecta millones de computadoras y dispositivos, permitiendo el intercambio de información de manera rápida y global.

Usos del internet

  • Buscar información
  • Comunicación
  • Educación
  • Entretenimiento
  • Trabajo

Protección en internet

La protección en internet consiste en aplicar medidas de seguridad para evitar riesgos como el robo de información, fraudes o acoso digital.

Medidas de protección

  • No compartir datos personales.
  • Usar contraseñas seguras.
  • No aceptar desconocidos en redes sociales.
  • Evitar enlaces sospechosos.
  • Usar antivirus.

Identidad digital

La identidad digital es la imagen que una persona crea en internet a través de sus publicaciones, comentarios, fotos y actividades en redes sociales.

Es importante cuidar la identidad digital porque influye en la reputación personal y profesional.

Recomendaciones

  • Publicar contenido responsable.
  • Respetar a los demás.
  • Configurar la privacidad.
  • Pensar antes de publicar.

La nube

La nube es un servicio que permite almacenar información en internet en lugar de hacerlo en la computadora.

Ejemplos de servicios en la nube

  • Google Drive
  • OneDrive
  • Dropbox

Ventajas de la nube

  • Acceso desde cualquier dispositivo.
  • Respaldo de información.
  • Ahorro de espacio.
  • Compartir archivos fácilmente.
